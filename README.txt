# RID Landing Page

Projeto separado em arquivos para abrir no Visual Studio Code.

## Estrutura
- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Como abrir
1. Extraia o arquivo zip.
2. Abra a pasta no Visual Studio Code.
3. Clique duas vezes em `index.html` ou use a extensão Live Server.
